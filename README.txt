This a Simple and Easy-to-Read C language implementation Library that
includes data structure and algorithm, etc.

INCLUDES:

  C.
  AVL-Tree, RB-Tree, Linked list, SwissTable.
  RingBuffer.
  AES, DES, Blowfish, ChaCha20, Salsa20, XChaCha20, XSalsa20.
  Bits stream (get/add).
  Base coding (16/32/64).
  JSON, INI.
  Big Number (4352/6400/8448).
  Blake2b, Blake2s.
  MD5, SHA1, SHA2 (224/256/384/512), SHA3.
  HMAC-Blake2b, HMAC-Blake2s, HMAC-MD5, HMAC-SHA2, HMAC-SHA3.
  HKDF-Blake2b, HKDF-Blake2s, HKDF-MD5, HMAC-SHA2, HKDF-SHA3.
  PBKDF2-Blake2b, PBKDF2-Blake2s, PBKDF2-SHA1, PBKDF2-SHA2, PBKDF2-SHA3.
  TOTP-SHA1, TOTP-SHA256, TOTP-SHA512.
  Bcrypt, Scrypt.
  SipHash24.
  Poly1305.
  xxHash (32/64).
  CRC (16/32/64).
  Deflate, Inflate.
  RSA (2048/3072/4096).
  Curve25519, Curve448.
  NIST-p256, NIST-p384, NIST-p521.
  ML-KEM (512/768/1024).
